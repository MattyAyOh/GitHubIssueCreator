HOW TO USE REPO-UNIQUE TEMPLATES:
  • Make sure there is a ISSUETEMPLATE.md file in the MASTER branch of your repository.
  • Put whatever markdown you would like in your template in that file
  • Indent everything in the template (4 spaces), so that it is recognized as "code"
  • !! Utilize "[]" around description fields. e.g "[description here]" !!
  • If the user clicks between "[]" or presses TAB, it will automatically select the entire field
